Themes

You can set the colour theme of the page by putting a {color}-theme class on the body on the page. You can use the selector in the top left to experiment with the different themes.

This also works on individual modules but is discouraged for anything othe than a front page.

Theme classes style all links, buttons and headers where appropriate for you.

The brand colours available are;
  • Pink
  • Red
  • Green
  • Blue
  • Purple
  • Yellow
  • Charcoal (Default)

Headers

Headers can be used in a 'traditional' way and will work perfectly. However, if you choose to use HTML5 sectioning (<section> or <article>), then you may use helper classes to achieve the same look.

Helper classes are: .heading-1 .heading-2 .heading-3 .heading-3-small and these can be applied to any heading.
